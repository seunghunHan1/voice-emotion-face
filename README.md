# 🎙️ Voice → Emotion → Face

목소리(운율)에서 감정을 추정해서, 사진 속 얼굴 표정을 그 감정에 맞게 바꾸는 **브라우저 단일 데모**입니다.
빌드 과정 없이 정적 파일만으로 동작합니다.

> 🔗 **라이브 데모**: (배포 후 갱신)

## 파이프라인

```
음성 → [운율 특징 추출] → [감정 추정] → [얼굴 표정 변형] → 바뀐 사진
       피치/에너지/말속도   휴리스틱 or LLM   AU 기반 메쉬 워핑
```

1. **음성 → 특징**: Web Audio API로 마이크 녹음 후, 자기상관 피치 추정 · RMS 에너지 · 피치 변동성 · 유성음 비율 · 말속도(근사) 추출
2. **특징 → 감정**: 두 가지 모드
   - **로컬 휴리스틱**: 특징을 valence(긍정도)·arousal(각성도)로 매핑 → 감정 라벨 (키 불필요, 즉시)
   - **OpenRouter LLM**: 음향 특징(+선택적 말 내용)을 LLM에 넘겨 추론 (본인 API 키 필요)
3. **감정 → 표정**: MediaPipe FaceLandmarker(468점) + Delaunay 삼각분할 → 감정별 Action Unit 근사 변위로 캔버스 메쉬 워핑

## 기술 스택

- 순수 HTML/CSS/JS (ES Modules, 빌드 없음)
- [MediaPipe Tasks Vision](https://developers.google.com/mediapipe) — 얼굴 랜드마크 (CDN)
- [Delaunator](https://github.com/mapbox/delaunator) — 삼각분할 (CDN)
- Web Audio API / MediaRecorder / (선택) Web Speech API
- (선택) [OpenRouter](https://openrouter.ai) — LLM 감정 추론

## 로컬 실행

마이크와 ES 모듈은 `localhost` 또는 `https`에서만 동작합니다. 정적 서버로 띄우세요.

```bash
cd voice-emotion-face
python3 -m http.server 8000
# 브라우저에서 http://localhost:8000 접속
```

### OpenRouter 키 자동 주입 (로컬 전용, 선택)

```bash
cp config.example.js config.local.js
# config.local.js 에 본인 OpenRouter 키를 넣으세요. (gitignore 대상 — 커밋 안 됨)
```

키가 없어도 휴리스틱 모드로 모든 기능이 동작합니다.

## 테스트 방법

페이지 상단 **"📖 사용법 & 테스트 방법"** 패널에 단계별 안내가 있습니다. 요약:

1. **사진 올리기** — ① 패널에서 정면 얼굴 사진 선택 → "얼굴 인식 완료" 확인
2. **표정만 테스트** — ③ 패널의 😊😢😠😮😨😐 버튼 + 강도 슬라이더로 즉시 확인 (음성 불필요)
3. **목소리 테스트** — ② 패널 `● 녹음 시작` → 2~4초 말하기 → `■ 녹음 중지` → 감정·표정 자동 반영

| 감정 | 표정 변화 포인트 |
|------|------------------|
| 😊 기쁨 | 입꼬리 위·바깥으로 |
| 😢 슬픔 | 입꼬리 내려가고 눈썹 안쪽 올라감 |
| 😠 분노 | 눈썹 안쪽이 아래·중앙으로 모임 |
| 😮 놀람 | 눈썹 올라가고 눈·입 벌어짐 |
| 😨 공포 | 눈썹 올라가며 모이고 눈 커짐 |
| 😐 중립 | 원본 복귀 |

## 보안 주의

- **공개 배포본에는 API 키를 넣지 않습니다.** 클라이언트 JS에 키를 박으면 누구나 탈취 가능합니다.
- OpenRouter 모드는 사용자가 UI에 직접 키를 입력해야 하며, 키는 해당 브라우저 `localStorage`에만 저장됩니다.

## 한계 (정직하게)

- 음성만으로 감정을 정확히 읽는 것은 본질적으로 불확실합니다. 실험실 정확도(70~80%)도 실제 환경에선 떨어집니다.
- 표정 변형은 신경망 생성이 아니라 **기하학적 근사(AU 기반 메쉬 워핑)**입니다. 큰 변형에서는 약간의 왜곡(seam)이 보일 수 있습니다.
- 정면·고해상도 얼굴 사진에서 가장 잘 동작합니다.
