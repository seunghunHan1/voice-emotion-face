📖 사용법 & 테스트 방법 (자세히)

처음 사용하는 순서

  1. 사진 올리기 — ①번 패널에서 정면 얼굴 사진을 선택합니다. "얼굴 인식 완료"가 뜨면 준비 끝. (배지에 "얼굴을 못 찾았어요"가 뜨면 더 정면·고해상도 사진으로 교체)
  2. 표정만 먼저 테스트 — ③번 패널의 😊😢😠😮😨😐 버튼을 눌러보세요. 음성 없이도 사진 표정이 즉시 바뀝니다. 강도 슬라이더로 변형 세기를 조절합니다.
  3. 목소리로 테스트 — 두 가지 방법:
    · ● 녹음 시작 → 2~4초 또렷하게 말하기 → ■ 녹음 중지 (마이크 권한 필요, https/localhost)
    · 🎵 음성 파일 첨부 → wav/mp3/m4a 등 오디오 파일 선택 (마이크·https 불필요, 어디서나 동작)
    추출된 운율 특징(피치·에너지·말속도)과 추정 감정이 ③번에 표시되고 사진 표정이 바뀝니다.

감정 인식 방식 두 가지

  • 로컬 휴리스틱(기본) — 키 불필요, 즉시 동작. 피치/에너지/말속도를 규칙으로 valence·arousal에 매핑.
  • OpenRouter LLM — ②번 "감정 인식 방식"을 펼쳐 API 키 입력. 음향 특징(+선택: 말 내용)을 LLM이 판단해 더 똑똑함. 공개 배포본에는 키가 없으니 본인 키를 직접 붙여넣어야 동작합니다(브라우저에만 저장).

감정별로 확인할 포인트 (수동 버튼 기준)

  • 😊 기쁨 — 입꼬리가 위·바깥으로 올라감
  • 😢 슬픔 — 입꼬리 내려가고 눈썹 안쪽이 올라감
  • 😠 분노 — 눈썹 안쪽이 아래·중앙으로 모임(찌푸림)
  • 😮 놀람 — 눈썹 전체 올라가고 눈·입이 벌어짐
  • 😨 공포 — 눈썹 올라가며 모이고 눈이 커짐
  • 😐 중립 — 원본으로 복귀

잘 안 될 때

  • 마이크 권한: 브라우저가 마이크를 요청하면 허용. 거부 시 녹음 불가.
  • 보안 컨텍스트: 마이크·모델 로딩은 https:// 또는 localhost에서만 동작합니다.
  • 얼굴 인식 모델은 최초 1회 CDN에서 내려받습니다(네트워크 필요).
  • 음성만으로의 감정 추정은 본질적으로 불확실합니다 — 표정은 기하학적 근사(AU 기반) 데모입니다.
